INTRODUCERE

În lumea tehnologiei și a dezvoltării software, Java a devenit unul dintre cele mai utilizate limbaje de programare. În cadrul acestui proiect, ne propunem să explorăm cum Java poate fi folosit nu doar pentru programare, ci și în analiza datelor. Vom investiga rolul său în manipularea și prelucrarea datelor, evidențiind natura sa orientată pe obiecte și adaptabilitatea în mediul de dezvoltare, precum IntelliJ, însoțit de JVM.

PROIECT OBIECTIV:

Obiectivul acestui proiect este de a demonstra și de a explora modul în care Java poate fi utilizată în două domenii distincte: analiza datelor și dezvoltarea de aplicații, evidențiind puterea sa de programare orientată pe obiecte.

ROLUL JAVA IN ANALIZA DATELOR:

Java oferă o suită variată de biblioteci și funcționalități pentru manipularea și analiza datelor. De la citirea și scrierea datelor în diferite formate până la prelucrarea lor în volume mari, Java poate fi folosită pentru a efectua operații complexe de analiză și transformare a informațiilor.

ABORDARE:

1.Manipularea Datelor: Se utilizeaza biblioteci Java precum Apache Commons CSV sau Jackson pentru manipularea și citirea datelor în diferite formate (CSV, JSON etc.).

2.Prelucrarea Datelor: Explorarea framework-urile precum Apache Hadoop sau Apache Spark pentru a efectua prelucrări complexe pe seturi mari de date.

3.Programare Orientată pe Obiecte: Evidentierea conceptele de OOP în Java, cum ar fi clasele, obiectele, moștenirea, încapsularea și polimorfismul.

4.Dezvoltarea Aplicației în IntelliJ și JVM: Crearea și rularea unei aplicații folosind conceptele și cunoștințele obținute, demonstrând modul în care Java funcționează în medii de dezvoltare integrate precum IntelliJ și cum este executată pe JVM.

REZULTATE ASTEPTATE:

Eficacitate in manipularea si analiza datelor,Intelegerea conceptelor de OOP si Dezvoltarea si rularea unei aplicatii.
